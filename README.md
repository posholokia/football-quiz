## Бекенд мобильного приложения футбольная викторина

### Настройка окружения
Директорию src отметить как Source root.

Настроить переменные окружения по примеру из example.env

### Запуск и остановка приложения

1. Запуск в докере:

`make run`

2. Остановить приложение:

`make down`
